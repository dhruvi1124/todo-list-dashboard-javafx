# To-Do List Dashboard Application

A JavaFX-based task management system developed for the Object Oriented Programming (OOP) Project Based Learning.

## Features
- Add tasks with description, priority, and due date
- Search tasks
- Sort tasks by priority
- Delete tasks
- Clear all tasks
- Progress tracking using progress bar
- Dark Mode / Light Mode support

## Technologies Used
- Java
- JavaFX
- Object Oriented Programming

## OOP Concepts Implemented
- Encapsulation
- Abstraction
- Modularity
- Event Driven Programming
